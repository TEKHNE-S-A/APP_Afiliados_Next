## 🔑 AUTENTICACIÓN

Al hacer `git push`, Git te pedirá credenciales:

### **Opción 1: Personal Access Token (RECOMENDADO)**
1. Ve a: https://github.com/settings/tokens/new
2. Dale permisos: `repo`, `workflow`
3. Copia el token (empieza con `ghp_`)
4. Al pedir contraseña: pega el token

### **Opción 2: SSH**
```powershell
# Generar clave
ssh-keygen -t ed25519 -C "tu@email.com"

# Agregar a GitHub: https://github.com/settings/ssh/new
# Copiar contenido de ~/.ssh/id_ed25519.pub
```

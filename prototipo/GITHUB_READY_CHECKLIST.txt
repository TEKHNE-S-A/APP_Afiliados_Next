## 🎯 FLUJO DE TRABAJO RECOMENDADO

### Para Nuevas Features:
1. `git checkout -b feat/nombre-feature`
2. Hacer cambios
3. `.\pre-push-validation.ps1`
4. `git add .` + `git commit`
5. `git push -u origin feat/nombre-feature`
6. Crear PR en GitHub

### Para Bug Fixes:
1. `git checkout -b fix/nombre-bug`
2. Corregir bug + tests
3. `git add .` + `git commit`
4. `git push -u origin fix/nombre-bug`
5. Crear PR en GitHub

### Branches Principales:
- **main**: Producción (protegida, solo PRs)
- **develop**: Desarrollo (rama base para features)
